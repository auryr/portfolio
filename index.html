My budget Manager

This app allows the users keep track on their expenses and their incomes. The users can create categories for the incomes and expenses, budgets for the desired period, and with that, they will they can compare what they have spent and what the were supposed to spend, which helps them to manage their finances.

HTML, CSS, JavaScript, Express, ReactJs, PostgreSql

Okurio

Okurio helps solve the problem of small team project management by aiming to keeping track of tasks among small groups of people. A project manager or administrator can create a profile that allows the creation of tasks/projects, the delegation of tasks, and the ability to keep track of collaborator activity.

HTML, CSS, JavaScript, Express, ReactJs, PostgreSql

Medieval Parcheesi

It's a version of the famous game "Parcheesi", but with a medieval touch. Each player has 4 warriors they should move depending on how many points they get through rolling the dices. The first player that gets their 4 warriors to the main kingdom wins, but there are certain rules and tricks in the game that won't make that so easy.

HTML, CSS, JavaScript, Jquery

Places to go

This Web app created for travelers, uses google places API and Google Images to find the places they should visit in a certain country or city. The users can create an user profile, where they can register the places they would like to visit, have a traveler journal and add notes to the places they have visited.

HTML, CSS, JavaScript, Express, PostgreSql
